# Robot Vision HUD

Мінімальна підсистема розпізнавання об'єктів для роботизованих систем.

## Що робить
- запускає YOLO на зображеннях, відео або вебкамері;
- фільтрує класи, релевантні для роботизованого середовища;
- формує робототехнічну інтерпретацію результатів;
- тестує стійкість до затемнення, пересвічування, шуму, розмиття та оклюзії;
- зберігає зображення з рамками та CSV-таблицю результатів.

## Швидкий запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Поклади тестові зображення у `examples/`.

Запуск:
python app.py

Режими:
IMAGE SCAN — обробка зображення
VIDEO STREAM — обробка відеофайлу
LIVE CAMERA — потік з камери
